# RGB通道混合器

一款用于将三张贴图导入并填充到新图片RGB三个通道的工具，适用于游戏开发中制作三合一贴图。

## 功能特点

- ✅ 支持将任意三张贴图导入并填充到新图片的RGB三个通道
- ✅ 点击空白预览区域选择图片
- ✅ 为每张图片独立分配R、G、B通道
- ✅ 创建高质量的混合图片
- ✅ 保存时强制转换为PNG格式
- ✅ 支持多种导出尺寸选择：512*512，1024*1024，2048*2048，4096*4096，8192*8192
- ✅ 现代化的外观设计，统一的白色背景
- ✅ 内容居中显示，界面美观
- ✅ 支持鼠标滚轮上下滚动
- ✅ 窗口大小自适应，内容自动调整

## 开发环境

- **操作系统**：Windows 10/11
- **Python版本**：3.13.9
- **开发工具**：任意Python IDE（如VS Code、PyCharm等）

## 依赖库

| 库名称 | 版本 | 用途 |
|--------|------|------|
| Pillow | 12.0.0 | 图片处理 |
| numpy | 2.3.5 | 通道混合计算 |
| pyinstaller | 6.17.0 | 软件打包（可选） |

## 安装步骤

### 1. 安装Python

从[Python官网](https://www.python.org/downloads/)下载并安装Python 3.13.9，安装时勾选"Add Python to PATH"。

### 2. 安装依赖库

打开命令提示符（CMD）或PowerShell，执行以下命令：

```bash
pip install pillow numpy
```

### 3. 运行软件

直接运行Python脚本：

```bash
python channel_mixer.py
```

## 使用说明

1. **选择图片**：点击界面中的空白预览区域，选择要导入的贴图
2. **分配通道**：为每张图片选择要填充的RGB通道
3. **创建混合图片**：点击"创建混合图片"按钮生成结果
4. **保存图片**：点击"保存图片"按钮，选择导出尺寸并保存为PNG格式

## 打包说明

如果需要将软件打包为可执行文件（.exe），可以使用PyInstaller：

### 1. 安装PyInstaller

```bash
pip install pyinstaller
```

### 2. 执行打包命令

```bash
python -m PyInstaller --onefile --windowed --name "RGB通道混合器" channel_mixer.py
```

### 3. 查找可执行文件

打包完成后，可执行文件将位于`dist`目录中：

```
dist/RGB通道混合器.exe
```

## 打包参数说明

- `--onefile`：生成单个可执行文件
- `--windowed`：隐藏控制台窗口
- `--name "RGB通道混合器"`：设置可执行文件名称

## 注意事项

- 支持的图片格式：PNG、JPG、JPEG、BMP、GIF
- 保存时默认使用PNG格式，确保图片质量
- 导出尺寸建议根据实际需求选择，较大尺寸会增加文件大小
- 软件在Windows 10及以上版本测试通过
